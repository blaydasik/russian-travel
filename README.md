# Учебный проект для изучения адаптивной вёрстки

## Обзор

Проект представляет собой одностраничный сайт, посвященный привлекательности путешествий по России. Включены ссылки на сторонние ресурсы.

## Проект создан для получения представления о способах адаптации веб страниц к просмотру на различных устройствах с разными размерами экранов. 

## Полученные навыки:
1. Практическое применение знаний о гридах;
2. Тренировка правильного применения и использования соотвествующих семантических тегов;
3. Умение использвать обе технологии (флексбокс и гриды) для решения различных задач и умение выбирать наиболее подходящую;
4. Использование псевдоэлементов в качестве решения нестандартной задачи;
5. Навыки по ведению одновременной разработки при помощи гит в разных ветках;
6. Навыки по умению своевременно и корректно делать бэкапы и вести разработку нового функционала в новых ветвях;
7. Умение работать с относительными размерами и величинами для реализации резиновой вёрстки;
8. Использование медиазапросов для реализации адаптивной вёрстки;
9. Умение подключать и использовать сторонние шрифты в проекте;
10. Тренировка навыков применения БЭМ;
11. Навыки работы с макетом в figma.

## Используемые технологии:
1. Гриды.
2. Флексбокс.
3. Медиазапросы;
4. Подход Desktop first;
5. Функция calc и относительные единицы измерения;
6. Оптимизация шрифтов;
7. Метатеги масштабирования;
